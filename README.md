# Environment

Ubuntu 20.04LTS  
ROS2 Foxy  

# Install

```bash
$ cd ~/ros2_ws/src
$ git clone https://github.com/porizou/map_server_ros2.git
$ cd ~/ros2_ws
$ colcon build
```

# Usage

```bash
ros2 run map_server_ros2 map_server_ros2
```
